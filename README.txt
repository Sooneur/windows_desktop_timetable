Программа для создания расписаний и заметок, отслеживания дедлайнов

Основное окно:
    Дата
    Текущие дедлайны на неделю (с цветоделением)
    Глобальные задачи (в порядке приоритета)
    Расписание на завтра
    Кнопка открытия работы с расписаниями,
    Кнопка info
Окно расписаний:
    Кнопки импорта, экспорта и создания расписания
    Расписание на текущий месяц
    Выбор изменяемого элемента расписания из списка
    Кнопка перехода для перехода в окно изменения расписания
Окно дедлайнов:
    Кнопки импорта, экспорта и создания дедлайнов
    Выбор изменяемого дедлайна
    Кнопка перехода в окно для редакции дедлайнов
Окно изменения:
    Поля для редактирования:
        Название
        Дата начала
        Период
        Конец
        Время
        Важность(Цвет)
        Заметки(картинки, описание)
Окно info:
    Немного о том зачем, я написал это и как стоит использовать приложение
